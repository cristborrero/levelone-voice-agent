💡 How ALEX Books Meetings
1. ALEX detects the intent to book via the LLM call_brain task
2. Queries Cal.com API for next available slots
3. Offers 2–3 time options to the caller verbally
4. Asks for email address, then creates the booking
5. Sends confirmation email via Resend, logs in HubSpot
